# Ejercicio 2.90

$$
f ( x ) = A \sin x + B x^2 + C x + D
$$

tiene tangente horizontal en el punto  $$ ( 0, 4 )$$   y además su derivada segunda es  $$ f'' ( x ) = 3  \tmop{sen}  x - 10$$  .
