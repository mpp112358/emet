# Ejercicio 2.79

 (Septiembre 2001, B.2, 2 ptos.) Sea  $$ P ( x )$$   un polinomio de grado 4 tal que:
 \begin{enumerate}
\item $P ( x )$ es una función par.
\item Dos de sus raíces son $x = 1$, $x = - \sqrt{5}$.
\item $P ( 0 ) = 5$.
\end{enumerate} 
Se pide:

A.    Hallar sus puntos de inflexión.

B.    Dibujar su gráfica.
