# Ejercicio 3.29

 (Junio 2005, A.2, 2 ptos.) Calcular un polinomio de tercer grado  $$ p
( x ) = ax^3 + bx^2 + cx + d$$   sabiendo que verifica:
 \begin{enumerateroman}
\item tiene un máximo relativo en $x = 1$.

\item tiene un punto de inflexión en el punto de coordenadas $( 0, 1
)$.

\item se verifica: $\int^1_0 p ( x ) d x = \dfrac{5}{4}$.
\end{enumerateroman}
