# Ejercicio 3.55

 (Septiembre 2007, A.4, 3 ptos.) Sea  $$ g ( x )$$   una función continua y
derivable para todo valor real de  $$ x$$  , de la que se conoce la
siguiente información:
 \begin{enumerateroman}
\item $g' ( x ) > 0$ para todo $x \in ( - \infty, 0 ) \cup ( 2, +
\infty )$, mientras que $g' ( x ) < 0$ para todo $x \in ( 0, 2 )$.

\item $g'' ( x ) > 0$ para todo $x \in ( 1, 3 )$ y $g'' ( x ) < 0$
para todo $x \in ( - \infty, 1 ) \cup ( 3, + \infty )$.

\item $g ( - 1 ) = 0, g ( 0 ) = 2, g ( 2 ) = 1$.

\item $\underset{x \rightarrow -\infty}{\lim} g ( x ) = - \infty$ y
$\underset{x \rightarrow + \infty}{\lim} g ( x ) = 3$.
\end{enumerateroman} 
Teniendo en cuenta los datos anteriores, se pide:

A.    Analizar razonadamente la posible existencia o no existencia
de asíntotas verticales, horizontales u oblicuas.


B.    Dibujar de manera esquemática la gráfica de la función  $$ g ( x
)$$  .


C.    Si  $$ G ( x ) =  \displaystyle \int ^x_0 g ( t ) d t$$   encontrar un valor  $$ x_0$$  
tal que su derivada  $$ G' ( x_0 ) = 0$$  .
